# Workshop 02 - Node.js HTTP-palvelin

Tämän workshop 2 tehtävän tarkoituksena on harjoitella node.js perusteita rakentamalla simppeli HTTP-palvelin. 

# Tavoitteet
Harjoituksen tavoitteena on ymmärtää:
- Miten HTTP-palvelin luodaan Node.jsällä
- Miten reititys toimii
- Miten staattisia tiedostoja (HTML, CSS, JS) tarjoillaan selaimelle
- Miten käsitellään virhetilanteita (404 ja 500)
- Miten luodaan yksinkertainen API-rajapinta
